# 🎬 Web Motion
> Create stunning motion graphics directly in your browser.

![Version](https://img.shields.io/badge/version-0.1.0-blue)
![Status](https://img.shields.io/badge/status-development-orange)
![License](https://img.shields.io/badge/license-MIT-green)

MotionFlow is a browser-based motion graphics and video editing application inspired by Alight Motion.

Create animations, edit videos, add effects, and export projects without installing any software.

---

## ✨ Features

### 🎞 Video Editing
- Import videos
- Trim and cut clips
- Multiple layers
- Timeline editor

### 🎨 Graphics
- Text layers
- Shape layers
- Image layers
- Custom fonts

### ⚡ Animation
- Keyframes
- Fade In / Fade Out
- Scale Animation
- Rotation Animation
- Position Animation

### 🎵 Audio
- Import audio files
- Audio timeline
- Volume control

### 📤 Export
- MP4 Export
- GIF Export (Planned)
- Project Save/Load

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/yourname/motionflow.git
```

### Open Project

```bash
cd motionflow
```

### Run

Use:

- VS Code
- Live Server Extension

Then open:

```text
http://localhost:5500
```
